Add File Uploads to Any Website

Using the Filestack JavaScript SDK, you can add a file picker to any HTML page with just a few lines of code.

JavaScript Quick Setup

Include the SDK

The fastest way to get started is to include the Filestack SDK script tag in your HTML file, right before the closing `</body>` tag.

<script src="//static.filestackapi.com/filestack-js/4.x.x/filestack.min.js"></script>

Initialize the Picker

Add a button to your HTML and write a script to initialize the Filestack client and open the picker when the button is clicked.


<!-- Add a button to your HTML -->
<button id="picker" class="bg-orange-500 text-white px-6 py-3 rounded-lg font-semibold hover:bg-orange-600">
  Upload File
</button>

<!-- Add your script -->
<script>
  const client = filestack.init('YOUR_API_KEY'); // Replace with your API Key
  const options = {
    onUploadDone: (res) => console.log(res.filesUploaded),
    // Add other picker options here
    maxFiles: 5,
    accept: ["image/*", "video/*"],
  };

  const pickerButton = document.getElementById('picker');
  pickerButton.addEventListener('click', () => {
    client.picker(options).open();
  });
</script>

Preview in Browser

There's no server needed. Simply save your code in an `.html` file and open it directly in your web browser.

Just open your HTML file to see it work!
