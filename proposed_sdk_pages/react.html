The Easiest Way to Upload Files in React

Integrate a powerful, beautiful, and easy-to-use file picker into your React application in minutes. This cheat sheet gets you started instantly.

React Quick Setup

Project Setup

First, ensure you have a React project created. Then, install the Filestack React SDK.

# Using npm
npm install filestack-react

# Using yarn
yarn add filestack-react

Implement the Picker

Import the `ReactFilestack` component and use it in your application. You'll need your Filestack API key.


import React from 'react';
import { ReactFilestack } from 'filestack-react';

const YourComponent = () => {
  const API_KEY = 'YOUR_API_KEY'; // Replace with your actual Filestack API key

  const handleSuccess = (result) => {
    console.log('Upload successful:', result);
  };

  const handleError = (error) => {
    console.error('Upload error:', error);
  };

  return (
    <div className="p-8 text-center">
      <h1 className="text-2xl font-bold mb-4">My Awesome App</h1>
      <ReactFilestack
        apikey={API_KEY}
        onSuccess={handleSuccess}
        onError={handleError}
        customRender={({ onPick }) => (
          <button 
            onClick={onPick} 
            className="bg-orange-500 text-white px-6 py-3 rounded-lg font-semibold hover:bg-orange-600"
          >
            Pick Files
          </button>
        )}
      />
    </div>
  );
};

export default YourComponent;

Start Your Application

Run your React development server to see the Filestack picker in action.

# Using npm
npm start

# Using yarn
yarn start

Your browser will open to `http://localhost:3000`, where you can launch the file picker.
