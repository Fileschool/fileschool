Storage Aliases

Integration

Powerful premium feature that bridges the Filestack platform with existing cloud storage infrastructure, allowing processing without data migration.

Integration Capability

Allows Filestack's Processing Engine to access, transform, and analyze files stored in external providers (Amazon S3, Azure Blob Storage) without uploading to Filestack's native storage.

URL Structure

.../API-KEY/src://STORAGE-ALIAS/PATH-TO-FILE

Business Benefits

  • Eliminates costly data migration
  • Preserves existing data lake investments
  • Enables gradual platform adoption
  • Decouples processing from storage

Related Terms

Processing Engine

Computational heart powering all file transformation and analysis tasks.

Handles

Unique alphanumeric identifiers assigned to every uploaded file.

File API

Foundational RESTful API providing programmatic control over stored file lifecycle.
