Document Viewer

Delivery Component

Component that extends delivery service to rich, in-app content consumption, rendering various document types directly in the browser.

Technology Foundation

Built upon Mozilla's PDF.js library, providing robust document rendering capabilities.

Supported Formats

  • PDF documents
  • Microsoft Office (Word, Excel, PowerPoint)
  • Adobe creative files (Illustrator, Photoshop)
  • Standard image and text files

User Features

  • Printing support
  • Text search functionality
  • Page rotation and zooming
  • Iframe embedding support

Premium Variant: Document Slider

Enhanced functionality with slideshow format presentation and ability to merge multiple documents into seamless presentations.

Related Terms

Delivery Service

Fast, secure, and reliable global distribution of assets.

File Conversion

Service that transforms files from one format to another.

Processing Engine

Computational heart powering all file transformation and analysis tasks.
