Domain Whitelists

Security

Broader, network-level security layer that complements fine-grained policy control by restricting service access to approved domains.

Operating Modes

  • Upload Whitelist: Restricts File Picker embedding by Origin header
  • Delivery Whitelist: Restricts asset embedding by Referer header

Protection Against

  • API key theft and unauthorized usage
  • Bandwidth theft through hotlinking
  • Unauthorized content distribution

Limitations

Headers can be spoofed by non-browser requests, so should be used as first line of defense alongside Security Policies.

Related Terms

Security Policies

Configuration rules that define access permissions and processing limitations.

File Picker

Filestack's customizable upload widget with multiple source options.

CDN (Content Delivery Network)

A distributed network of servers used by Filestack to deliver files and images with optimized performance.
