Virus & Malware Detection

Security/AI

Fundamental security feature that scans uploaded files for malicious content, available exclusively as asynchronous Workflow task.

Architectural Design

Exclusively asynchronous to avoid impacting user experience during potentially time-consuming scans.

Response Structure

  • "infected": Boolean field indicating malware presence
  • "infections_list": Array containing detected malware names

Workflow Integration

Conditional logic examples: infected eq true triggers quarantine or deletion actions.

Use Cases

  • Educational platforms scanning student submissions
  • Enterprise file sharing security
  • User-generated content protection

Related Terms

Workflows

Automated processing sequences with multiple operations and conditional logic.

Intelligence (AI) Service

Machine learning models for content analysis and understanding.

Security Policies

Configuration rules that define access permissions and processing limitations.
