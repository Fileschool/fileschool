Image & Video Tagging

AI/Content Understanding

Machine learning feature that automatically analyzes visual content to extract meaningful keywords, transforming unstructured media into structured, searchable data.

Analysis Output

Returns detected objects and concepts with numerical confidence scores indicating model certainty.

Business Applications

  • E-commerce: Automated product attribute assignment
  • Media Companies: Video archive cataloging
  • Content Discovery: Searchable media libraries

Processing Model

Available as asynchronous Workflow task, suitable for bulk content processing.

Example Output

Video analysis might return: "animal" (0.95), "cheetah" (0.87), "wildlife" (0.82)
