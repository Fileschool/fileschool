Handles

Core Component

The unique, alphanumeric identifier assigned to every file upon successful upload to the Filestack platform, serving as the primary key throughout the file's lifecycle.

Handle Architecture

Functions as the central concept that links all of Filestack's services together, providing a single identifier for complete platform access.

Universal Usage

  • CDN delivery URLs
  • Processing API transformations
  • File API operations
  • Security policy references

Developer Benefits

Dramatically simplifies API surface area - developers only need to store this single string to gain full access to all Filestack capabilities for that file.

Related Terms

File API

Foundational RESTful API providing programmatic control over stored file lifecycle.

Processing API

URL-based API that handles file transformations and processing operations.

CDN (Content Delivery Network)

A distributed network of servers used by Filestack to deliver files and images with optimized performance.

Security Policies

Configuration rules that define access permissions and processing limitations.
