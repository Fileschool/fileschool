Delivery Service

Core Service

Responsible for the fast, secure, and reliable global distribution of assets processed by the Filestack platform.

CDN Infrastructure

Built upon a high-performance Content Delivery Network leveraging Fastly CDN with Anycast routing and global points of presence (POPs).

Performance Features

  • Global edge caching
  • Automatic optimization
  • Progressive JPEG support
  • Responsive image delivery

Advanced Features

  • Document Viewer for in-browser rendering
  • Custom CNAME for branded URLs
  • Seamless Transformations integration

Related Terms

CDN (Content Delivery Network)

A distributed network of servers used by Filestack to deliver files and images with optimized performance.

Custom CNAME

Premium feature enabling custom domain branding for asset URLs.

Document Viewer

In-browser rendering component for multiple file types.

Transformations

Real-time file processing operations without permanent storage of intermediate versions.
