Drag and Drop Upload

User Interface

User interface feature that allows files to be uploaded by dragging them from the file system directly into a web application using Filestack's upload widgets.

Implementation

Built into the File Picker widget, providing an intuitive upload experience without requiring users to navigate through file dialogs.

Features

  • Visual drop zone indication
  • Multiple file selection support
  • Progress indicators during upload
  • Error handling and validation

Browser Support

Supported across modern browsers with automatic fallback to traditional file selection methods when needed.

Related Terms

File Picker

Filestack's customizable upload widget with multiple source options.

Upload Widget

Customizable UI component for file selection and upload functionality.

File Upload API

RESTful API endpoints that handle file upload operations.
