Optical Character Recognition (OCR)

AI/Data Extraction

Intelligence service that unlocks information within images by detecting and extracting both printed and handwritten text.

Availability

  • Synchronous: Processing API task for immediate results
  • Asynchronous: Workflow task for bulk processing

Structured Output

Highly structured JSON with extracted text plus precise bounding box coordinates for each text area, line, and individual word.

Advanced Applications

  • Document digitization and indexing
  • License plate recognition
  • Product information extraction
  • Security footage text analysis

Coordinate Data

Enables applications beyond text extraction, supporting layout analysis and targeted content manipulation.

Related Terms

Intelligence (AI) Service

Machine learning models for content analysis and understanding.

Document Detection

AI-powered feature that automatically identifies and processes document types.

Processing API

URL-based API that handles file transformations and processing operations.

Workflows

Automated processing sequences with multiple operations and conditional logic.
