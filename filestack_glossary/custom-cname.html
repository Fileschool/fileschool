Custom CNAME

Enterprise Feature

Premium, enterprise-grade feature allowing organizations to brand their Filestack asset URLs with their own domain.

Branding Benefits

Instead of cdn.Filestackcontent.com, organizations can use custom subdomains like cdn.fs.YOUR-FQDN for seamless white-labeled experience.

Security Enhancements

  • Security policies stored in browser cookies vs URL parameters
  • Policy information not visible in shared URLs
  • Enhanced access control opacity
  • Reduced URL complexity

Implementation

Requires DNS configuration and is available as part of enterprise plans with enhanced security requirements.
