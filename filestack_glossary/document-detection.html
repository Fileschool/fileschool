Document Detection

AI/Intelligence

AI-powered feature that automatically identifies and processes document types, including skew correction and format optimization for uploaded documents.

Detection Capabilities

  • Automatic document type identification
  • Skew correction for scanned documents
  • Format optimization
  • Text region identification

Integration

Available through the Intelligence Suite and can be incorporated into Workflows for automated document processing pipelines.

Use Cases

  • Document digitization projects
  • Automated document classification
  • OCR preprocessing
  • Quality improvement for scanned documents

Related Terms

Optical Character Recognition (OCR)

Intelligence service extracting printed and handwritten text from images.

Intelligence (AI) Service

Machine learning models for content analysis and understanding.

Workflows

Automated processing sequences with multiple operations and conditional logic.

File Conversion

Service that transforms files from one format to another.
