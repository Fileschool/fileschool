Safe for Work Detection

Check if images are suitable for workplace environments using AI-powered content analysis.

Configuration

About Safe for Work

The Safe for Work (SFW) feature uses advanced AI to analyze images and determine if they are appropriate for workplace environments. It can detect:

  • Adult content
  • Violence
  • Drugs and alcohol
  • Hate symbols
  • Inappropriate text

How it works

The SFW API returns a confidence score indicating how likely the image is to be inappropriate for work environments. Lower scores indicate safer content.
