Size Calculation Tips

  • 1 MB = 1,048,576 bytes (binary) or 1,000,000 bytes (decimal)
  • 10 MB = 10,485,760 bytes (good for documents)
  • 100 MB = 104,857,600 bytes (good for high-res images)

  • Dimensions: Always use format width,height (e.g., 1920,1080)

Advanced API Methods & Controls

Programmatic control over the drag & drop instance after initialization.

Element Management

Dynamically add, remove, or replace drop zones after initialization:

  • dnd.setElements(elements) - Replace all drop zones with new elements
  • dnd.setElement(element) - Replace all with a single element
  • dnd.addElement(element) - Add one more drop zone to existing list
  • dnd.addElements(elements) - Add multiple drop zones to existing list
Example: 
dnd.addElement(document.querySelector('.another-drop-zone'));

Upload Control

Pause, resume, or cancel uploads programmatically via events:

  • dnd.emit('pause', {elementId}) - Pause all uploads from an element
  • dnd.emit('pause', {fileId}) - Pause a specific file upload
  • dnd.emit('resume', {elementId}) - Resume paused uploads
  • dnd.emit('cancel', {elementId}) - Cancel uploads completely
Example: 
dnd.emit('pause', { elementId: 'drop-zone-1' });

Helper Properties

Access internal helpers for advanced control:

  • dnd.elementHelper - Direct access to element manager (elements array)
  • dnd.eventEmitterHelper - Event emitter for custom events
  • dnd.uploadsHelper - Upload manager for configuration changes
  • dnd.filstackSdk - Full Filestack Client instance access
Example: 
// Open file picker from same SDK instance
dnd.filstackSdk.picker(options).open();

Quick Tips & Best Practices

  • Visual Feedback: Style your drop zone with CSS hover effects and drag-over indicators
  • Accept Patterns: Use MIME types like image/* for broader file matching
  • Size in Bytes: Remember: 1MB = 1,048,576 bytes, 10MB = 10,485,760 bytes
  • Multiple Zones: Use setElements() to manage multiple drop zones efficiently
  • Event Listeners: Use .on('uploadFileFinish', callback) to track completed uploads
  • Security: Add policy/signature for private uploads or server-side file restrictions
  • CDN + SRI: Use Subresource Integrity (SRI) hashes from manifest.json for security

About Safe for Work Detection

Safe for Work (SFW) detection is an AI-powered intelligence service that analyzes image content and returns a boolean value indicating whether the image contains appropriate content for workplace environments.

How It Works

The AI model analyzes visual content including objects, people, context, and scenes to determine if the image contains inappropriate content such as adult content, violence, or other NSFW material. The analysis happens in real-time and returns instant results.

Response Format

Safe Image: {"sfw": true}
Unsafe Image: {"sfw": false}

Simple boolean response makes it easy to implement conditional logic in your application
  • Real-Time Processing - Synchronous API with instant results
  • Workflows Compatible - Can be used in automated workflows
  • Chainable - Combine with other transformations in a single URL
  • Security Required - Needs policy & signature for protection
  • Universal Support - Works with handles, external URLs, and storage aliases

Common Use Cases

About OCR

Optical Character Recognition (OCR) is an AI-powered intelligence service that extracts both printed and handwritten text from images and documents. It returns detailed JSON with extracted text, bounding box coordinates, line-by-line analysis, and word-level detection.

How It Works

The AI model analyzes the document to detect text areas, then breaks them down into lines and individual words. Each element includes bounding box coordinates (x, y positions) allowing you to locate and extract specific text regions. Works with printed text, handwriting, images, PDFs, and scanned documents.

Response Format

{ "document": { "text_areas": [{ "text": "Extracted text", "bounding_box": [{x, y}], "lines": [{ "text": "Line text", "words": [{ "text": "word", "bounding_box": [...] }] }] }] }, "text": "Full extracted text", "text_area_percentage": 23.4 }

Returns hierarchical text structure with precise coordinates
  • Text Extraction - Extract printed & handwritten text
  • Coordinate Detection - Get bounding boxes for all text
  • Multi-Language - Supports multiple languages
  • Hierarchical Data - Text areas, lines, and words
  • Universal Support - Images, PDFs, and documents
  • Security Required - Needs policy & signature
